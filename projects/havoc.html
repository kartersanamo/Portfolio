About

Havoc is a Factions-era plugin for Spigot 1.8.8 with procedurally placed bases that can be raided, a breach/salvage flow backed by an in-world currency, loot tables, and timed terrain/claim resets.

Features

  • Procedural bases — generates raidable structures programmatically across the world.
  • Loot tables — configurable drops for chests and raid rewards.
  • Salvage currency — custom in-world currency used in shops and salvage flows.
  • Admin tools — in-plugin admin commands and a shop GUI (`/havoc shop`).

Integrations

Compiles against WorldEdit 6.1.x and uses SaberFactions at runtime via a small reflection bridge (`FactionsBridge`).

Commands

Primary command: `/havoc`. Admin entrypoint: `/havoc admin`. Shop: `/havoc shop`.

Why it matters

Havoc provides emergent PvP content for Factions servers with automated base placement, predictable reset mechanics, and customizable rewards to keep gameplay fresh.
