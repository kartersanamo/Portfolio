Tag Central

Tag Central is a desktop Python app for managing vessel tags across Proficy and Cimplicity. It provides import dry-run previews, Cimplicity link reports, an export queue for prepared Proficy CSVs, merge & alignment tools, internal mismatch grouping, and automatic backups.

Python Proficy · Cimplicity Pandas (on-demand) Featured project
View on GitHub Read the README

Project summary

This repository contains utilities and tools related to tag generation, normalization, and aggregation. Visit the repository for the full README and examples.

What I built

  • Import analyzers - Proficy and Cimplicity importers with dry-run previews that surface matches, ambiguous rows, and suggested link actions.
  • Cimplicity link report - a post-import breakdown that matches by PT_ID, address, alias, and collects rows that require manual review.
  • Export queue - assemble, review, edit, and validate pending Proficy CSV exports grouped per vessel before writing files for re-import.
  • Internal mismatch grouping - detect groups (G/A/P) of tags with similar descriptions or addresses to help reconcile duplicates and near-duplicates.
  • Tag merge & align - merge two tags (one survives, one queued for Proficy export) and align descriptions to Cimplicity when appropriate.
  • Backups & auto-backup - manual backup/restore and automatic backups before large bulk operations to prevent data loss.
  • Desktop packaging - scripts and instructions to build platform installers (.app, .exe) and regenerate icons for branded releases.

Typical workflow

  1. Import Proficy — run a dry-run preview then apply changes or resolve conflicts.
  2. Import Cimplicity — preview link statistics, resolve sync dialog items, and review the link report.
  3. Cimplicity review — create Proficy tags for unmatched Cimplicity points (queued for export).
  4. Review export queue — verify pending Proficy rows grouped by vessel before export and optionally validate CSVs.
  5. Export Proficy changes — write per-vessel batch CSVs for re-import into Proficy.
  6. Complete Cimplicity tasks — perform manual HMI edits and mark tasks done in the manual task checklist.

Why it matters

Consistent tag identifiers make search, categorization, and automation more reliable across multiple projects and tools.

How to use

Technical notes

The app uses a single canonical CSV (`tags.csv`) as the tag database and stores supporting JSON files such as `cimplicity_review_queue.json`, `cimplicity_manual_tasks.json`, and `alias_rules.json`. Exports are written beside the app in an `Exports/` folder and backups are kept under `backups/` in the app data location (`~/Library/Application Support/TagCenter/` on macOS or `%APPDATA%\TagCenter\` on Windows). Pandas is loaded only when importing/exporting Excel files to keep startup fast.

Result: Tag Central centralizes tag reconciliation across Proficy and Cimplicity workflows: Cimplicity acts as the source of truth when linked, while Proficy is updated via validated export CSVs and manual task lists.
