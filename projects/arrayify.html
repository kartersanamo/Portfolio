Arrayify

Convert tank register spreadsheets into array-based rows, apply DOCX soundings, and normalize register names to tank tags. Lightweight CLI and optional Tkinter GUI.

Python CSV Tkinter NEW
View on GitHub README

About

Arrayify is a small utility written to automate conversions between tank register spreadsheets for ECO. It refactors single-address sounding variables into arrays, can apply new sounding values from .docx tables, and normalizes register names into tag-style identifiers.

How it works

  • Arrayify — consolidates contiguous register addresses into array columns for compact, consistent spreadsheets.
  • Sounding — reads `.docx` tables and applies sounding values to matching rows.
  • Normalization — converts free-form register names to normalized tag names for downstream tooling.
  • Interfaces — runs via a CLI menu or an optional Tkinter GUI; designed to be packaged with PyInstaller.

Design

Workflows are implemented in `tank_tools.services` as single-responsibility classes: `ArrayifyService`, `TankSoundingService`, and `TagNormalizationService`.

Usage

Run `python main.py` for the CLI menu or `python main.py --gui` to launch the Tkinter interface. The tool uses only the Python standard library at runtime.

Why it matters

Arrayify saves manual spreadsheet cleanup and reduces errors when producing tank register outputs for downstream systems. The optional sounding step integrates values from DOCX tables so workflows stay reproducible.
